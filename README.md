# 言值 — 口才训练小程序

> 让每一次开口，都有价值

## 项目概述

「言值」是一款基于 UniApp 开发的微信小程序，专注于帮助用户通过 AI 驱动的个性化口才训练，提升职场表达、面试应答、社交沟通等场景的表达能力。

## 核心功能

- 🎤 **语音录制与 AI 分析** — 6维度评分（逻辑/流畅度/填充词/语音语调/感染力/时长）
- 📚 **场景化训练** — 工作汇报/面试/即兴发言/商务谈判/社交破冰/公开演讲
- 🎯 **技能训练** — PREP/SCQA/STAR 表达框架、填充词控制、语音语调等
- 📊 **成长数据面板** — 能力雷达图、趋势曲线、成就徽章
- 🏆 **口才等级体系** — Lv.1-10 游戏化成长路径
- 💡 **每日技巧** — 表达框架、演讲技巧、实战建议
- 👑 **会员体系** — 免费版获客 + 会员版变现

## 技术栈

- **前端框架**: UniApp (Vue 3)
- **目标平台**: 微信小程序
- **语音采集**: 微信 RecorderManager API
- **语音识别**: 腾讯云 ASR (待接入)
- **AI 分析**: 大模型 API (Qwen/GLM) — 当前使用 mock 模拟
- **数据存储**: 本地 Storage (生产环境需接入后端)

## 快速开始

### 安装 HBuilderX

1. 下载 [HBuilderX](https://www.dcloud.io/hbuilderx.html)
2. 导入本项目文件夹

### 运行项目

1. 在 HBuilderX 中选择 `运行 → 运行到小程序模拟器 → 微信开发者工具`
2. 首次运行会自动安装依赖

### 或使用命令行

```bash
# 安装依赖
npm install

# 微信小程序开发
npm run dev:mp-weixin

# 微信小程序打包
npm run build:mp-weixin
```
